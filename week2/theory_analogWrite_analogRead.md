# analogRead()
 analogRead reads a voltage from the analog pins only.
 Returns values between 0 and 1023.
 Example: Reading data from an LDR or soil moisture sensor.

# analogWrite()
  It can be used on any PWM pin and create PWM wave.
  Accepts values between 0 and 255.
  Example: Controlling LED brightness or DC  motor speed.

 # PWM
   Pulse Width Modulation, or PWM, is a technique for getting analog results with digital means 
   by rapidly switching a digital pin ON and OFF. 
   It is used to control the brightness of LEDs, motor speed, and fan speed efficiently.
