# LED Blink Project

## Project Title
LED Blink Project

## Hardware Required
- Arduino Uno
- LED
- 220Ω Resistor
- Jumper Wires
- USB Cable

## Circuit Diagram Description
- Connect the anode of the LED to Digital Pin 13 through a 220Ω resistor.
- Connect cathod) of the LED to the GND pin.
- 
## How to Upload Code
1. Open Arduino IDE.
2. Connect the Arduino Uno to the computer using a USB cable.
3. Write the code into the Arduino IDE.
4. Click the **Verify** button to compile the code.
5. Click the **Upload** button to upload the code to the Arduino board.

## Expected Output
- The LED connected to Pin 13 blinks continuously.
- The Serial Monitor displays the blink count.
- 
## Troubleshooting Tips
1. Ensure the LED is connected with the correct polarity.
2. Check that the resistor is connected properly.
3. Verify all jumper wire connections before uploading.
