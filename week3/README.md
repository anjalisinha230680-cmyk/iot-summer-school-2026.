# Environmental Monitoring Station (Q21)

Features:
- Reads temperature in °C and °F
- Reads humidity
- Updates every 2 seconds
- Prints data in CSV format
- Red LED ON if temperature > 35°C or humidity > 80%
- Green LED ON when conditions are normal

Library Used:
- Adafruit DHT Sensor Library v1.4.6
