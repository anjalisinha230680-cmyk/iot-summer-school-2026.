# Q13. Arduino Pin Types

## 1. Digital Input
**Explanation:**  
Digital input pins are used to read two states: HIGH (1) or LOW (0).

**Example:**  
Push button or motion sensor.

**Real IoT Use Case:**  
A smart door system detects when a button is pressed to unlock the door.

---

## 2. Digital Output
**Explanation:**  
Digital output pins send HIGH or LOW signals to control devices.

**Example:**  
LED or relay module.

**Real IoT Use Case:**  
A smart home system turns an LED or light ON and OFF remotely.

---

## 3. Analog Input
**Explanation:**  
Analog input pins read variable voltage values (0–1023).

**Example:**  
LDR (Light Dependent Resistor) or temperature sensor.

**Real IoT Use Case:**  
A smart street light measures surrounding light and turns ON automatically at night.

---

## 4. PWM Output
**Explanation:**  
PWM (Pulse Width Modulation) pins simulate analog output by varying the pulse width.

**Example:**  
LED brightness control or DC motor speed control.

**Real IoT Use Case:**  
An automatic fan controller adjusts fan speed based on room temperature.

---

## 5. I2C/SPI Pins
**Explanation:**  
I2C and SPI are communication protocols used to connect Arduino with sensors and other devices.

**Example:**  
OLED display, LCD, or MPU6050 sensor.

**Real IoT Use Case:**  
A weather monitoring system sends data from sensors to an OLED display using I2C communication.
