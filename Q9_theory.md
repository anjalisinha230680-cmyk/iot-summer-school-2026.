## Difference between git pull,git fetch, and git clone.and when to use.
**git clone**
It is used when a copy of existing git repository to your local computer.
when to use: copying a repository

**git fetch:**  
It downloads the latest changes from the remote repository but does not merge them into your current branch. 
It is used to check updates safely.
when to use : when you want to see updates without affecting your code

**git pull:**  
It downloads the latest changes from the remote repository and automatically merges them into your current branch.
It is used to update your local repository.
when to use: when you want to update your code with latest changes
